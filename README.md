# Git Test Project



```shell
1.git init
2.git add
3.git commit -m "备注"
4.git config user.name "hunter"
5.git config user.email "xxxxx" 
6.git log
7.git status
8.git add .  //提交所有到缓存
9.git restore main.go // 缓存回退
10.git restore -- staged README.md // 本地回退
11.git reset 45c17ff //版本回退 到45c17ff版本
12.git reflog //查询关联日志 
13.git remote add origin http://github.com/wei-rh/gitTest.git 
14.git remote -v // 查看
15.git remote remove origin //删除origin
16.git push -u origin master //master分支 
17.git clone github.com/wei-rh/....  // 远程创建本地库

```
